# STACKIT CMF Cloud Foundry Example

This repository contains the Terraform baseline for the CMF cloud-foundry example.

## Usage

1. Copy `env.tfvars.example` to `terraform.tfvars`.
2. Fill in the required variables.
3. Run `terraform init`.
4. Run `terraform apply`.

## Features

- Cloud Foundry workload deployment (Spring Music)
- Configurable modules via feature flags
